Available Commands

Here are the main commands you can use with Hyper Bot.

/ban [user]

Bans the specified user from the server.

/kick [user]

Kicks a member from your server.

/mute [user] [time]

Mutes a user for a specific amount of time.

/joke

Tells a random joke to lighten the mood.

/ping

Returns the bot's current latency.

/userinfo [user]

Displays information about a user.
